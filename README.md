
# 🌐 MERN Social Media App

A full-stack MERN (MongoDB, Express, React, Node.js) social media application with authentication, posts, likes, and file uploads.
Built with JWT authentication, Redux Toolkit, MUI, and Multer for image handling.




## ✅ Backend

- User Authentication (Register/Login with JWT)
- Password encryption using bcrypt
- Image upload via Multer
- CRUD operations for users and posts
- Like/Unlike functionality for posts
- MongoDB models for Users and Posts
- Secure API with helmet and CORS handling

## ✅ Frontend
- React with Redux Toolkit for state management
- Redux Persist for storing state in localStorage
- Material-UI (MUI) for UI components
- React Router v6 for navigation
- Formik + Yup for form handling & validation
- Image upload using react-dropzone
- Authentication flow with JWT token persistence
## 🛠️ Tech Stack

**Frontend:** React, Redux Toolkit, Redux Persist,    React Router v6, Material-UI, Formik, Yup, React Dropzone


**Backend:** Node.js, Express, MongoDB, Mongoose, Multer, JWT, Bcrypt

**Other Tools:** dotenv, helmet, morgan, cors
